# FimaApp

## Prerequisite

Create a secrets.json file in /assets with your Airtable API_BASE_ID and API_KEY

```
-lib/
-assets/ 
  |- icon/
  |- secret.json

```

**secrets.json**

```json
{
	"AIRTABLE_API_BASE_ID": "<your_api_base_id>",
	"AIRTABLE_API_KEY": "<your_api_key>"
}
```

## Run

```sh
flutter run
```
